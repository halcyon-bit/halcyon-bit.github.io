Redis

Redis
数据库
发布日期:   2022-06-26
文章字数:   1.5k
阅读时长:   6 分

一、NoSQL

NoSQL：一类新出现的数据库（not only sql）

  • 泛指非关系型数据库；
  • 不支持 SQL 语法；
  • 存储结构跟传统关系型数据库中的那种关系表完全不同，NoSQL 中存储的数据都是 KV 形式；
  • NoSQL 的世界中没有一种通用的语言，每种 NoSQL 数据库都有自己的 API 和语法，以及擅长的业务场景；
  • NoSQL 中的产品种类：Redis、Mongodb 等。

二、特性

  • Redis 支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用；
  • Redis 不仅仅支持简单的 key-value 类型的数据，同时还提供 list，set，zset，hash 等数据结构的存储；
  • Redis 支持数据的备份，即 master-slave 模式的数据备份。

三、数据类型

Redis 默认有 16 个数据库，通过 0-15 标识，默认选择第一个数据库。select 10

  • 字符串 string
  • 哈希 hash
  • 列表 list
  • 集合 set
  • 有序集合 zset

key 不能重复

3.1 string

字符串类型是 Redis 中最基础的数据存储类型，它在 Redis 中是二进制安全的，这意味着该类型可以接受任何格式的数据，如 JPEG 图像数据或 json 数据等。在 Redis 中字符串类型的 Value 最多可以容纳的数据长度是 512M。

3.1.1 添加、修改

如果设置的键不存在则为添加，如果已存在则为修改。

语法：set key value

set name hello
set name world

Redis 的 key 可以设置过期时间，到期后该键值对失效。

语法：setex key seconds value

setex name 5 hello

设置多个键值对

语法：mset key1 value1 key2 value2 ...

mset name1 hello name2 world

3.1.2 获取

根据键获取值，如果不存在则返回 nil

语法：get key

set name hello
get name

根据多个键获取多个值

语法：mget key1 key2 ...

mset name1 hello name2 world
mget name1 name2

3.1.3 删除

删除键及对应的值

语法：del key1 key2 ...

set name hello
del name

3.2 hash

hash 是一个 string 类型的 field（字段）和 value（值）的映射表，hash 特别适合用于存储对象。

3.2.1 添加、修改

语法：hset key field1 value1 field2 value2 ...

hmset key field1 value1 field2 value2 ... 

hset name a 1 b 2 c 3 d 4
hset name a 5 b 2 c 3 d 4

3.2.2 获取

获取指定键的所有属性

语法：hkeys key

hkeys name

获取指定键的一个属性的值

语法：hget key field

hget name a

获取指定键的多个属性的多个值

语法：hmget key field1 field2

hmget name a b

获取指定键的所有属性的值

语法：hvals key

hvals name

3.2.3 删除

删除整个键值对，使用 del 命令。

删除指定键的某些属性

语法：hdel key field1 field2

hdel name a b

3.3 list

list 是简单的字符串列表，按照插入顺序排序。可以添加一个元素到列表的头部（左边）或者尾部（右边）

3.3.1 增加

左侧插入数据

语法：lpush key value1 value2 ...

lpush name 1 2 3
内存中：3 2 1

右侧插入数据

语法：rpush key value1 value2 ...

rpush name 4 5 6
内存中 4 5 6

3.3.2 获取

返回列表里指定范围内的元素

语法：lrange key start stop

start、stop 为元素的下标索引，索引从左侧开始，第一个元素为 0；索引可以是负数，表示从尾部开始计数，如 -1 表示最后⼀个元素。

lrange name 0 -1

3.3.3 删除

删除整个键值对，使用 del 命令。

删除指定键的某些元素。

语法：lrem key count value

根据参数 count 的值，移除列表中与参数 value 相等的元素。

  • count > 0：从表头开始向表尾搜索，移除与 value 相等的元素，数量为 count
  • count < 0：从表尾开始向表头搜索，移除与 value 相等的元素，数量为 count 的绝对值
  • count = 0：移除表中所有与 value 相等的值
lpush name 1 2 3 4 5 6 1 2 3 1 1 1
内存：1 1 1 3 2 1 6 5 4 3 2 1
lrem name 3 1
内存：3 2 1 6 5 4 3 2 1
lrem name -1 2
内存：3 2 1 6 5 4 3 1

3.4 set

set 是 string 类型的无序集合，集合成员是唯一的。集合是通过哈希表实现的，所以添加，删除，查找的复杂度都是 O(1)。

3.4.1 增加

语法：sadd key member1 member2

sadd name 1 2 3 1 4 2
内存：1 2 3 4

3.4.2 获取

获取所有元素

语法：smembers key

smembers name

3.4.3 删除

删除整个键值对，使用 del 命令。

删除指定键的某些元素。

语法：srem key member1 member2 ...

srem name 1 2

3.5 zset

zset 是 string 类型的有序集合，集合成员是唯一的。每个元素都会关联一个 double 类型的分数。redis 是通过分数来为集合中的成员进行从小到大的排序。分数(score)是可以重复。

3.5.1 增加

语法：zadd key score1 member1 score2 member2 ...

zadd name 1 a 2 b 3 c 1 d
内存 a d b c

3.5.2 获取

语法：zrange key start stop

start、stop 的使用与 list 相同。

zrange name 0 -1

3.5.3 删除

删除整个键值对，使用 del 命令。

删除指定键的某些元素。

语法：zrem key member1 member2

zrem name a b

3.6 其他命令

3.6.1 查找键

支持正则表达式

语法：keys pattern

keys *
keys na*

3.6.2 判断键是否存在

语法：exists key1 key2 key3

若 key 存在返回 1，否则返回 0。

exists name

3.6.3 查看键对应 value 的类型

语法：type key

type name

3.6.4 键的过期时间

设置过期时间，以秒为单位。

语法：expire key seconds

expire name 10

查看有效时间，以秒为单位。

语法：ttl key

ttl name
